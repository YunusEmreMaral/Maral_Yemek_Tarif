# Maral Yemek Tarif

## Projenin Amacı
- Kullanıcılar ve Adminin birlikte iletişim kurarak yemek tarifleri paylaşması. 
- İnsanlara yeni yemekler öğretmek .
- Admin paneli desteği ile 1 yerden yönetim.

## Kullanılan Teknolojiler , Kütüphaneler ve Yöntemler
- HTML
- CSS
- ASP.NET
- C#
- MSSQL
